_____________________

若两点相等，斜率为：λ = (3Xp² + a)/2Yp

若两点不相等，则斜率为：λ = (Yq - Yp)/(Xq - Xp)

计算结果点R的坐标计算公式为：

Xr = (λ² - Xp - Xq) 

Yr = (λ(Xp - Xr) - Yp)

伪造函数
______________

如果只需要签名消息的Hash值，那么任何人都可以伪造签名。

Ecdsa验证旨在验证：

$s^{-1}$(eG+rP) = (x',y')=R',r'=x' mod n == r

为了伪造，我们可以选择u,v,计算R'=(x',y')=uG+vP，之后选择r'=x' mod n，计算：

e'=r'uv' mod n

s'=r'v' mod n

伪造即可完成
